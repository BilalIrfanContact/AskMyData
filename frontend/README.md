# AskMyData Frontend

Production frontend for AskMyData, with Supabase Google auth and backend API integration.

## Setup

```bash
cd frontend
npm install
npm run dev
```

## Env vars (`frontend/.env.local`)

```bash
VITE_API_BASE_URL=http://localhost:8000
VITE_SUPABASE_URL=...
VITE_SUPABASE_ANON_KEY=...
```
